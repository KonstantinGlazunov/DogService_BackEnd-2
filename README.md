## Настройка проекта

* Необходимо обеспечить наличие Environment Variables
  * Для базы данных:
    * `DB_HOST`
    * `DB_PORT`
    * `DB_NAME`
    * `DB_USERNAME`
    * `DB_PASSWORD`
  * Для отправки писем:
    * `base_url`
    * `spring_mail_password`
    * `spring_mail_username`
  * Для работы с Digital Ocean/AWS S3
    * `s3_accessKey`
    * `s3_endpoint`
    * `s3_region`
    * `s3_secretKey`
